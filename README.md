# rules-us-ar

Arkansas tax and benefit rules encoded in Cosilico DSL.

## Structure

```
rules-us-ar/
├── statutes/       # Arkansas Code (A.C.A.) provisions
└── regulations/    # Arkansas agency regulations
```
